# Capstone 3
This project focuses on analyzing sales data for the Emporium Bookstore using PowerBI. My goal is to transform raw data into meaningful and easy to understand visualizations.

My Dashboard highlights key sales trends according to the geographic distribution of the data.

To effectively visualize my data I have employed the use of:
* Line Chart
* Pie Chart
* Bar Chart
* List
